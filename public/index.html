US States REST API

INF653 Back End Web Development - Final Project

Endpoints

  • GET /states/ — all state data
  • GET /states/?contig=true|false — contiguous / non-contiguous
  • GET /states/:state — data for one state
  • GET /states/:state/funfact — random fun fact
  • GET /states/:state/capital | nickname | population | admission
  • POST /states/:state/funfact — add fun fact(s)
  • PATCH /states/:state/funfact — replace by index
  • DELETE /states/:state/funfact — remove by index
